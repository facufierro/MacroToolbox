# Hotkey Manager v1.0.5

## Changes

- Fixed hotkeys that press a modifier key (Ctrl, Shift, Alt, or Win — including their left/right variants) failing to run: the generated AutoHotkey script had a syntax error that prevented it from loading, disabling every hotkey in the profile.
- The "hold" action now keeps its key(s) held down for as long as you hold the hotkey, then releases them — so a hotkey can act as a held modifier or remap (e.g. making a forced Copilot key behave as Ctrl). Previously "hold" only tapped the key, the same as "press".
