# Hotkey Manager v1.6.5

## Changes

- Fixed hold-to-repeat crashing on the first press (an uninitialized timer variable), which sent a key down but never released it and left a key stuck down. The key-up is now always sent, so a press can never strand a key held.
- Fixed edits to the global scope not taking effect — its always-on script now regenerates whenever you change it (previously it only regenerated on app launch, so newly added global hotkeys, e.g. Copilot/Right-Ctrl accents, did nothing until a restart).
