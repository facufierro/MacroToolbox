# Hotkey Manager v1.4.0

## Changes

- Added a `repeat` hotkey step that auto-fires a key at a chosen interval (in milliseconds) for as long as the trigger is held, stopping the moment you release it. The repeated key supports modifiers just like `press` and `hold`.
- The "Enable Hotkeys Key" no longer defaults to backtick (`` ` ``). Leave it empty to bind no key — previously an all-scopes scope silently captured the backtick key everywhere, even with the overlay disabled or a separate overlay key set.
- Fixed "Launch on startup" not taking effect on Windows; the app now reliably registers itself to run at login.
