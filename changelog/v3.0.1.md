# MacroToolbox v3.0.1

## Changes

- Fixed the **"Unable to uninstall!"** error when installing an update while MacroToolbox was running. Because the app runs elevated and lives in the tray, its previous version was still running and locked its own files, so the update couldn't replace them. The installer now automatically closes the running app before updating and installs with administrator rights, so updates apply cleanly.
- As part of this, MacroToolbox now installs **for all users** (into Program Files) and asks for an administrator confirmation once when installing or updating.
