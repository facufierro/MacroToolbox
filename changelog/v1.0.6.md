# Hotkey Manager v1.0.6

## Changes

- Fixed hotkeys (most noticeably the global profile in the tray) going dead after the PC slept or sat idle for a while: the hotkey script now reinstalls its keyboard/mouse hooks when Windows wakes up or silently drops them, and the app force-restarts the script after a resume from sleep. Recovery now triggers off the keyboard hook itself, so moving the mouse no longer hides a dead keyboard hook — hotkeys come back as soon as you start using the keyboard again.
- Added a "Launch on startup" option in Settings that registers Hotkey Manager to start automatically when you log in to Windows. Combine it with "Open to tray" to start hidden in the tray.
