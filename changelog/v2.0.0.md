# MacroToolbox v2.0.0

## Changes

- Renamed the app from "Hotkey Manager" to "MacroToolbox" to reflect its broader scope beyond game hotkeys — running actions in response to events and adjusting behaviors across any application. The window title, tray menu, and startup entry now show the new name.
- Because the app's internal identifier changed with the rename, settings and hotkeys saved by "Hotkey Manager" are not carried over automatically. Re-import or re-create your configuration after updating.
