# Hotkey Manager v1.6.4

## Changes

- Hold-to-repeat now lets you set a **key-hold time** (how long each press holds the key down) alongside the interval. Some games act on a key every frame it is held, which made repeats fire far faster than the interval you set; each press now holds the key for a short, precise, adjustable window (default 6ms) so it registers as a single press. Lower the hold if a game still fires too fast, raise it if presses don't register. Existing repeats keep working and pick up the default.
