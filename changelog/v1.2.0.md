# Hotkey Manager v1.2.0

## Changes

- Fixed remapping a key combo that includes modifiers (such as the Copilot key, which sends LWin+LShift+F23) to a held key: the trigger's own modifiers are now released, so the remap outputs only the target key (e.g. a clean RCtrl) instead of leaking LWin+LShift through underneath it.
- Recording a hotkey trigger now captures keys that Windows reserves for itself — most notably the Copilot key — instead of opening Copilot. While you're recording, the key you press is intercepted and captured (including the full LWin+LShift+F23 combo) rather than being handled by Windows.
- Global-scope single-key remaps (e.g. binding the Copilot key to RCtrl) are now applied directly at the system level instead of through AutoHotkey: the source key truly becomes the target key everywhere, with no leftover modifiers leaking through and without ever opening Copilot. These remaps are always active and aren't affected by the hotkeys enable/disable toggle.
