# Hotkey Manager v3.2.2

## Changes

- Fixed update checks not running when the window is shown from the tray, which could suppress update notifications indefinitely.
- Fixed sent keys landing on the wrong key on layouts that move letters around (e.g. QWERTZ, where "send y" pressed the physical Z key): characters now resolve through the active layout, with the US-position fallback kept for layouts like Cyrillic where the character doesn't exist.
