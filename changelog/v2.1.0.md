# MacroToolbox v2.1.0

## Changes

- Added a Scripts tab to each scope for running Python scripts. A script runs either when its hotkey is pressed (while the scope's app is focused) or when the scope's app is launched, and its body can be typed in as inline Python code or pointed at a `.py` file on disk. Each script can be enabled/disabled and run on demand with a "Run now" button for testing.
- Added a "Python Executable Path" setting to choose which Python interpreter runs scripts. Leave it empty to use `python` (falling back to the `py` launcher) from your PATH.
