# Hotkey Manager v3.2.4

## Changes

- Fixed hotkeys feeling laggy or "cold-starting" on the first press after the app had been idle in the tray: the background AutoHotkey process is now kept out of Windows power throttling so keypresses dispatch promptly.
- Removed a hidden per-keypress delay: hotkeys that don't drive an overlay state no longer make a blocking localhost request on every press, so accents, autocast, and other plain hotkeys fire without waiting on the overlay backend.
